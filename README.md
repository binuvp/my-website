# Dr. Binu V P - Personal Website

A professional, responsive personal website for Dr. Binu V P, featuring a modern glassmorphism design.

## Project Structure
- `index.html`: Main content file.
- `style.css`: All visual styles and glassmorphism effects.
- `script.js`: Navigation logic and scroll animations.

## How to Run Locally
1. Clone the repository or download the files.
2. Open `index.html` directly in your web browser.
   - Or, for a better experience, use a local server (e.g., VS Code "Live Server" extension).

## Hosting on GitHub Pages
1. Create a new repository on GitHub (e.g., `my-website`).
2. Upload `index.html`, `style.css`, and `script.js` to the root of the repository.
3. Go to **Settings** > **Pages**.
4. Under **Source**, select `main` (or `master`) branch and `/root` folder.
5. Click **Save**.
6. Your site will be live at `https://<username>.github.io/my-website/`.
